What is alphabinary encoding?

It's binary, written using letters of the alphabet, where letters from the first half are zeroes, and letters from the second half are ones:

For example, the word "WISDOM" equates to the binary value 101010, or 42 (in decimal), while the word "NEXUS" (10111) equates to 23 (as do other words, such as pawns, setup, Venus, tarot, plots, and Nixon).

Alphabinary encoding can be used to encode longer values as well; for instance, the MD5 hash sum of the Universal Declaration of Human Rights (as encoded by the Unicode Consortium):

can be encoded alphabinarily as "Teaspoonfuls unconsciously ordained preferable vilification enabling hoodwinking skeletons consequences trumping misfortunes parallel plague", a 140-character statement just short enough to fit in a Tweet. Such a hashsum could then be used to verify integrity out-of-band, or to obtain the document from a distributed hash table.

In general, the homomorphism afforded by alphabinary encoding opens the door for a lot of intriguing possibilities: for now, check out the GitHub organization.
