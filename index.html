What is alphabinary encoding?

It's binary, written using letters of the alphabet, where letters from the first half are zeroes, and letters from the second half are ones:

For example, the word "WISDOM" equates to the binary value 101010, or 42 (in decimal), while the word "NEXUS" (10111) equates to 23 (as do other words, such as pawns, setup, Venus, tarot, plots, and Nixon).

This homomorphism opens the door for a lot of intriguing possibilities: for now, check out the GitHub organization.
